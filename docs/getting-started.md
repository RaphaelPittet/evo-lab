# 🚀 Getting Started

Everything you need to set up your environment and run any experiment in this repo.

---

## Requirements

- Python 3.11+
- pip
- Windows / macOS / Linux

---

## Install Python

Download from [python.org](https://www.python.org/downloads/).

During installation on Windows:
- ✅ Check **"Add Python to PATH"**
- ✅ Click **"Disable path length limit"** at the end

---

## Clone the repo

```bash
git clone https://github.com/YOUR_USERNAME/evo-lab.git
cd evo-lab
```

---

## Install dependencies

Each experiment lists its own dependencies in its README.  
Common ones across all experiments:

```bash
pip install pygame numpy
```

---

## Run an experiment

```bash
cd 01-natural-selection-basics
python main.py
```

Refer to each experiment's own README for controls, parameters, and expected output.
